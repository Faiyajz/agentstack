@AGENTS.md

## Codex

AgentStack can be used as a Codex skill collection. The active workflows live in `skills/<name>/SKILL.md`.

- Prefer shared behavior in `AGENTS.md`; keep Codex-specific adapter notes here.
- Invoke skills by their standalone names when available: `product-brief`, `design-doc`, `spec`, `plan`, `implement`, `tdd`, `fullstack-feature`, `backend-api`, `frontend-ui`, `devops-deploy`, `server-admin`, `security-review`, `incident-response`, `interview-prep`, `coding-interview`, `system-design-interview`, `behavioral-interview`, `technical-article`, `dsa-teacher`, `refactor`, `review`, `address-pr-feedback`, `browser-verify`, `branch`, or `commit`.
- `browser-verify` requires a browser-capable tool such as Chrome DevTools MCP or a project Playwright fallback.
- Do not duplicate shared workflow rules here.
