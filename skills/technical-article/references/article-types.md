# Article Types

Classify the article before drafting.

| Type | Use When | Example Required |
|---|---|---|
| Tutorial | Teaching a workflow from start to finish | Usually yes |
| How-to | Solving one practical problem | Usually yes |
| Deep dive | Explaining internals, tradeoffs, or design | Sometimes |
| Comparison | Choosing between tools or approaches | Sometimes |
| Architecture guide | Designing a system or pattern | Diagram or example |
| Debugging guide | Diagnosing and fixing failures | Reproduction preferred |
| Case study | Explaining a real project or decision | Evidence preferred |
| Release/news analysis | Explaining a recent change | Usually no |
| Opinion | Arguing a position | Usually no |

Planning output should include:

- article type
- reader level
- reader promise
- central thesis
- example type and scope
- research questions
- done criteria
