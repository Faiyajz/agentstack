# Translation

Translate only when requested.

Rules:

- Preserve code blocks, command names, file paths, API names, and identifiers.
- Translate comments only when it helps the target reader and does not break code.
- Preserve source links and frontmatter relationships.
- Keep technical terms in the original language when that is the normal usage.
- Re-review translated headings, examples, and callouts for meaning.

Output translations as `<slug>.<lang>.md`.
