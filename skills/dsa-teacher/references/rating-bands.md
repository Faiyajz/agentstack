# Rating Bands

Use this as a teaching map, not a claim about official platform ratings.

## Codeforces-Style Ratings

| Rating | Learner Band | Teaching Focus |
|---:|---|---|
| 800-900 | Beginner | implementation, simple math, direct observations |
| 1000-1100 | Beginner+ | sorting, counting, greedy starts, simple strings |
| 1200-1300 | Pupil | two pointers, prefix sums, basic constructive, casework |
| 1400-1500 | Pupil+ | binary search, greedy proof, stack, graphs basics |
| 1600 | Specialist entry | DP basics, harder greedy, graph/state reasoning |

## LeetCode Mapping

LeetCode does not expose official Codeforces-style problem ratings. When the user asks for LeetCode 800-1600:

- Use Easy as roughly 800-1100.
- Use Medium beginner as roughly 1200-1400.
- Use Medium stronger-pattern problems as roughly 1500-1600.
- Use a third-party rating only when the user provides or requests a source.

Always label this as an approximate learning map, not official LeetCode metadata.
