# Practice Set Format

Use this when the user asks to categorize a range or produce similar questions.

```markdown
# Practice Set: <Range or Pattern>

## Goal

What the learner should be able to do after finishing.

## Recommended Order

| # | Problem | Platform | Approx Rating/Difficulty | Pattern | Why This Problem |
|---:|---|---|---|---|---|

## Pattern Groups

### <Pattern Name>

- What this pattern means
- When to recognize it
- Common mistakes
- Problems in this group

## 20 Similar Questions

Group by easy to hard. Each item should include:

- problem name
- platform
- rating/difficulty if known
- core pattern
- one-line learning value

## How To Study

1. Try for 20-30 minutes.
2. Read only the key observation if stuck.
3. Code from memory.
4. Dry run on custom tests.
5. Re-solve after 3-7 days.
```

Do not fabricate ratings. Use approximate labels when exact ratings are unavailable.
